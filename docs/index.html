What I can do for you

Accessibility & heuristic reviews.


Struggling to manage WCAG compliance as your product scales?

I can do a detailed or high-level accessibility / heuristic review, and based on the outcomes suggest fixes and next steps.

Front-end development.


As a front-end developer and designer I can wireframe, prototype and build products that are user friendly, accessible and scalable.

I can build accessible websites and prototypes, make improvements to existing products, or work to implement new features.

Wireframing & prototyping.


Want to test ideas before building them?

I can create one-off or batch prototypes using Figma that you can share with your team or with users to test.

Creating design systems.


I build good looking, user-friendly and responsive design systems using the Atomic Design Methodology. I can run the process from start to finish, and design directly to code.

I enjoy challenges involving different ways of working and visualising data and content, and have a lot of fun writing creative CSS / JS.
